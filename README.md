# CustomAuth Swift Samples

Examples of using [Torus CustomAuth Swift SDK].

**Usage**

1. Clone the repository and open with Xcode

2. Run the app, you'll see a demo app demonstrating full functionalities
   of [Torus CustomAuth Swift SDK]

- OAuth/Social logins

- Aggregate logins returning same keys for different providers

- Native/custom logins using `getTorusKey` and `getAggregateTorusKey`

3. All configurations are written to `Auth.plist`, update the values accordingly to try
   with your specific configurations.
